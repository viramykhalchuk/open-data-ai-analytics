# open-data-ai-analytics

## Назва і мета
Мета проєкту — завантажити набір відкритих даних з порталу data.gov.ua, 
перевірити якість даних, виконати базове дослідження та побудувати 
візуалізації для отримання аналітичних висновків.

## Джерело відкритих даних
Портал відкритих даних України: https://data.gov.ua/  
(посилання на конкретний набір даних буде додано після вибору датасету)

## Питання / гіпотези для аналізу
1. Як змінюється ключовий показник у часі (по днях/місяцях/роках), чи є 
сезонність або тренд?
2. Чи є суттєві відмінності між категоріями/регіонами (якщо вони присутні 
в наборі даних)?
3. Чи існує зв’язок між двома основними змінними набору 
(кореляція/залежність), і які фактори найбільше впливають на показник?
